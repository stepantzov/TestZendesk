Preconditions
A Zendesk Sell Free trial account.

Test scenario
-Log into the Zendesk Sell web application (using the Trial Account).
-Create a new Lead. Check that the Lead status visible on the details page is set to "New"
-Go to Settings → Leads → Lead statuses and change the name of the "New" status to some different name
-The status name change is reflected in the Lead details page
